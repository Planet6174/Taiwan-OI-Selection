15. `n` 为非负整数，加减乘除和大小比较需要常数等级的时间。根据下面的程序，哪一个描述是正确的？
```c
long foo (long x, long n)
{
  long f;
  if (n % 2 == 0)
    f = 1;
  else
    f = x;
  if (n < 2)
    return f;
  return f*foo(x*x, n/2);
}
```
a. `foo(x, n)` 会返回 `nx`  
b. `foo(x, n)` 会返回 `xn`  
c. `foo(x, n)` 的时间复杂度是 $\Theta(\log x)$  
d. `foo(x, n)` 的时间复杂度是 $\Theta(n \log n)$
